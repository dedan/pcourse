control structures
==================

**for** loop
------------

* when to use
    * iterate over something
    * when you know from beginning how often to do something

examples

    for i in range(10):
        print i

    import glob
    for file_name in glob.glob('*.txt'):
        print file_name


**if** condition
----------------

* when to use
    * when a decision has to be made

examples

    if age < 18:
        print 'too hard for you'
    else:
        print 'come in'
